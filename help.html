HOW TO SEARCH PROTEIN SEQUENCES?

  • Enter the species name and protein name in the input fields.

    Click the search icon to start the search.

  • Important:

    Both the species name and protein name must be entered

    otherwise, the search cannot proceed

  • Once the target sequence is found, a tracking ID will be generated .

    Keep this tracking ID for future referenceâ€”it allows you to revisit your results.

    You can download the sequence in FASTA format, or proceed with further analysis using the retrieved sequence.

  • If an incorrect species or protein name is entered, the system will not find any matching sequences.

    You will need to adjust your input and try again.

HOW TO PERFORM SEQUENCE ANALYSIS?

  1. There are three types of analyses available:

    Conservation Analysis: Evaluates evolutionary conservation among sequences.

    Motif Search: Identifies known motifs within the sequences using the PROSITE database.

    Phylogenetic Tree Construction: Reconstructs evolutionary relationships using IQ-TREE.

  2. How to Start an Analysis:

    Upload your own sequence file (must be in FASTA format, the system will validate the file).

    Alternatively, enter a tracking ID from a previous search and click the search icon to start the analysis.

  3. Once the analysis is complete, you can download the results as a ZIP file and copy the tracking ID by clicking id button.

    If an error occurs during the analysis, an alert will be shown, and you will need to reload the page to retry

  • extra-information of conservation analysis:

    The system will display the conservation plot, showing functionally important sequence regions

  • extra-information of motif analysis:

    The motif analysis results will indicate the number of sequences containing known motifs and their names

  • How to visualise phylogenetic tree:

    1. The generated Newick-format tree file can be downloaded.

    2. Click the "cat" button to download the tree file.

    3. Use the iTOL (Interactive Tree of Life) website to visualise and explore the phylogenetic tree.

HOW TO TRACK PREVIOUS SEARCH?

  • If you have a tracking ID (unique id start with seq_)

    enter it in the search bar and click the search icon to retrieve previous results

  • If the tracking ID is not found,

    The system will prompt you to enter a valid ID

  • if no analysis was performed

    The system will display the original sequence file

    allowing you to download it or proceed with further analysis

  • If you have performed any of the three analyses,

    your original sequence file will be displated alongside the analysis resutls

    The results can be downloaded as a ZIP file (click "result_link" button) for further examination

    Additional information will be provided.
